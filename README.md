# 墨迹 - 个人博客

## 博客介绍

"墨迹"是一个个人技术博客，记录技术探索与生活感悟，分享Python编程技巧、前端开发经验和个人思考。

### 博客特点

- **现代化设计**：采用响应式布局，适配不同屏幕尺寸
- **视觉效果**：3D旋转头像、粒子动画、打字机效果、点击涟漪特效
- **交互体验**：平滑滚动、悬停效果、主题切换
- **功能完备**：文章列表、文章详情、文章管理
- **数据持久化**：使用本地存储存储文章数据

## 技术栈

- **前端框架**：Vue.js 2
- **样式**：Tailwind CSS
- **字体**：Cormorant Garamond + Noto Sans SC
- **动画**：CSS动画 + JavaScript动画
- **数据存储**：localStorage

## 如何使用

### 1. 浏览文章
- 打开首页，查看最新文章列表
- 点击文章标题进入文章详情页
- 点击"返回列表"按钮返回文章列表

### 2. 管理文章
- 在浏览器控制台输入 `localStorage.setItem('isAdmin', 'true')` 开启管理员模式
- 刷新页面，导航栏会出现"管理"按钮
- 点击"管理"按钮进入文章管理页面
- 在管理页面可以添加、编辑、删除文章
- 所有修改会自动保存到本地存储

### 3. 切换主题
- 点击右下角的主题切换按钮
- 可以在浅色和深色主题之间切换

### 4. 控制音乐
- 点击左下角的音乐控制按钮
- 可以播放或暂停背景音乐

## 数据管理

- **数据存储**：文章数据存储在浏览器的本地存储中
- **初始数据**：首次加载时会从 `data/articles.json` 文件导入数据
- **数据备份**：建议定期导出本地存储数据进行备份
