# 墨迹个人博客 - 项目规格说明

## 1. 项目概述

### 1.1 项目名称
**墨迹 (JUweb)** - 个人技术博客

### 1.2 项目描述
一个现代化的个人技术博客网站，记录技术探索与生活感悟，采用 Vue.js + Tailwind CSS 构建，包含丰富的视觉效果和交互体验。

### 1.3 主要功能
- 博客文章展示（列表 + 详情）
- 文章管理（增删改查）
- 深色/浅色主题切换
- 背景音乐播放控制
- 丰富的视觉特效（粒子动画、赛博朋克风格、点击特效等）

### 1.4 目标用户
- 技术博客读者
- 博主本人（管理文章）

## 2. 技术架构

### 2.1 技术栈
| 类别 | 技术选型 | 版本/来源 |
|------|----------|-----------|
| 前端框架 | Vue.js | 2.6.14 (CDN) |
| CSS框架 | Tailwind CSS | 3.x (CDN) |
| 字体 | Cormorant Garamond | Google Fonts |
| 字体 | Noto Sans SC | Google Fonts |
| 动画 | 原生JavaScript + CSS | Canvas API |
| 数据存储 | localStorage + JSON | 浏览器本地存储 |

### 2.2 文件结构
```
JUweb/
├── index.html              # 博客首页（主版本）
├── admin.html              # 文章管理页面（主版本）
├── css/
│   └── style.css           # 主样式文件
├── data/
│   └── articles.json       # 文章数据
├── html/
│   ├── index.html          # 博客首页（子目录版本）
│   └── admin.html          # 文章管理页面（子目录版本）
├── js/
│   ├── app.js              # Vue应用核心逻辑
│   └── script.js          # 特效和工具函数
└── media/
    ├── bg.mp4              # 背景视频
    ├── bgyy.mp3            # 背景音乐
    └── touxiang.png        # 头像
```

## 3. 页面规格

### 3.1 首页 (index.html)

#### 3.1.1 导航栏
- 固定顶部，毛玻璃背景效果
- 左侧：Logo "墨迹"（点击返回首页/顶部）
- 右侧：管理员模式下显示"管理"链接

#### 3.1.2 Hero 区域
- 全屏高度
- 视频背景（移动端禁用）
- 粒子动画覆盖层
- 背景纹理
- **根目录版本特效**：
  - 赛博朋克霓虹标题 "CODE"
  - 全息投影头像 + 旋转光环
  - 粒子组成文字动画
- **子目录版本特效**：
  - 3D旋转立方体头像
  - 打字机效果标题 "记录思考"
- 底部滚动提示箭头

#### 3.1.3 文章列表
- 最多7篇文章/页
- 文章卡片包含：
  - 分类标签
  - 发布日期
  - 文章标题
  - 文章摘要
  - 标签列表
- 分页导航组件

#### 3.1.4 文章详情
- 毛玻璃容器
- 返回列表按钮
- 文章标题 + 内容（支持HTML）
- 响应式排版样式

#### 3.1.5 关于我模块
- 个人简介（两段文字）
- GitHub联系方式链接

#### 3.1.6 页脚
- 年份自动更新
- 标语 "用文字丈量世界"

### 3.2 管理页 (admin.html)

#### 3.2.1 权限控制
- 检查 `localStorage.isAdmin === 'true'`
- 非管理员重定向到 index.html

#### 3.2.2 文章列表区
- 显示所有文章
- 每项包含：标题、日期、分类
- 操作按钮：编辑、删除

#### 3.2.3 文章编辑表单
- 字段：标题、分类、标签（逗号分隔）、摘要、内容
- 支持添加和编辑两种模式
- 保存到 localStorage

## 4. 功能规格

### 4.1 文章系统

#### 4.1.1 数据结构
```javascript
{
  id: Number,           // 文章ID
  title: String,        // 标题
  excerpt: String,      // 摘要
  date: String,         // 日期 (YYYY-MM-DD)
  category: String,     // 分类
  tags: String[],       // 标签数组
  content: String        // HTML内容
}
```

#### 4.1.2 数据加载逻辑
1. 优先从 localStorage 读取
2. 若无，从 `data/articles.json` 加载
3. 首次加载后保存到 localStorage

### 4.2 主题切换
- 支持深色主题（默认）
- CSS变量定义颜色方案
- 主题切换按钮（仅子目录版本）

### 4.3 音乐控制
- 左下角固定音乐控制按钮
- 播放/暂停切换
- 循环播放
- 点击按钮控制

## 5. 视觉效果规格

### 5.1 视频背景
- 固定定位，z-index: -2
- 自动播放、循环、静音
- 移动端移除

### 5.2 粒子动画
- Canvas 实现
- 粒子数量：50个
- 鼠标规避效果
- 粒子间连线（距离<150px）

### 5.3 赛博朋克效果
- 霓虹发光文字
- 故障叠加层（红/青双色）
- 闪烁动画

### 5.4 全息投影头像
- CSS实现扫描线效果
- 旋转双光环
- 色调偏移滤镜

### 5.5 点击特效
- 涟漪扩散动画
- 随机颜色、大小、速度
- 可选发光效果

### 5.6 像素光标轨迹
- 鼠标移动生成像素点
- 随机彩虹颜色
- 1秒后消失

## 9. 已知限制

1. **无后端**：所有数据存储在浏览器 localStorage
2. **无搜索**：文章数量少，未实现搜索功能
3. **无评论**：未集成评论系统
4. **重复文件**：根目录和 html/ 子目录有重复页面
5. **移动端视频**：自动禁用背景视频
