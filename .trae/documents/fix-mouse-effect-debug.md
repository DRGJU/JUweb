# 鼠标特效问题排查与修复计划

## 问题分析

用户报告在桌面端看不到新的鼠标特效。可能的原因：

1. **DOMContentLoaded 事件冲突** - 存在两个事件监听器，可能导致初始化顺序问题
2. **Canvas 被遮挡** - z-index 可能仍然不够高
3. **初始化函数未被正确调用** - 可能有 JavaScript 错误

## 修复步骤

### 步骤 1：合并 DOMContentLoaded 事件监听器
- 将第29行和第487行的两个事件监听器合并为一个
- 确保所有初始化函数在一个监听器中按正确顺序调用

### 步骤 2：确保 Canvas z-index 正确
- 确认 Canvas 的 z-index 为 99999（最高）
- 确保 `pointer-events: none` 以免阻挡点击

### 步骤 3：添加调试日志
- 在 `initAdvancedCursorEffect()` 函数开头添加 console.log
- 帮助确认函数是否被正确调用

### 步骤 4：检查浏览器控制台错误
- 提示用户打开 F12 控制台检查错误

## 预期结果
- 鼠标移动时显示星光拖尾
- 鼠标点击时显示粒子爆发
- 鼠标周围显示呼吸光环
